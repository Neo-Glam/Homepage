-Portfolio – Première version responsive

Ce dépôt contient le code source de mon portfolio personnel. Il s'agit de la première version entièrement responsive, développée afin de présenter mon profil, mes compétences ainsi que une sélection de projets réalisés au cours de mon apprentissage en développement web.

- Objectifs du projet

Concevoir un site simple, clair et professionnel.

Appliquer les principes fondamentaux du responsive design.

Améliorer ma maîtrise de HTML, CSS et des techniques modernes de mise en page.

Disposer d’un support en ligne pour présenter mon travail.

- Technologies et outils

HTML – Structure et sémantique

CSS – Mise en forme, animations, responsive design

CSS Grid – Gestion de la mise en page

Media Queries – Adaptation aux différents formats d’écran


- Responsive Design

Le site a été conçu pour s’adapter automatiquement aux principaux formats d’affichage :

Mobile (smartphones)

Tablettes

Écrans desktop

Grands écrans

- Fonctionnalités principales

Section dédiée aux projets

Navigation fluide et intuitive

Design propre et minimaliste

Adaptation complète aux différents supports
